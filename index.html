? What is a Computer

A computer is an electronic device that performs calculations and processes data according to code instructions

Components of a Computer

  • CPU (Central Processing Unit)
  • RAM (Random Access Memory)
  • Motherboard
  • Storage (HDD/SSD)
  • Power Supply Unit (PSU)
  • Graphics Card (GPU)
  • Cooling System

Assembly Steps

  1. Prepare your workspace and gather all components.
  2. Install the CPU onto the motherboard.
  3. Insert RAM modules into the motherboard slots.
  4. Mount the motherboard into the case.
  5. Connect the power supply to the motherboard and components.
  6. Install storage drives (HDD/SSD).
  7. Add the graphics card if applicable.
  8. Connect all cables and peripherals.
